      PDF取り込み（自動解析）

      問題・解答・解説が載った PDF をアップロードすると、AI が大問ごと・セクションごとに自動で振り分けます。解析後に内容を確認・修正してから登録してください。
      ※ Anthropic API キー（接続設定タブ）と Worker URL が必要です。

      追加プロンプト（AI変換ルール）

      PDF取り込みの自動解析に、独自の変換ルールを追記できます。記法（空所 [[ ]]・選択肢 (())・語注 ##語::訳##・下付き ~~x~~ など）に沿った指示を書くと精度が上がります。設定は Worker（D1）に保存され全端末で共有されます。

      登録データの一括置換（grep replace）

      すでに登録済みの全問題（問題・解答・解説の本文）に対し、上から順に文字列を置換します（例：「，」→「、」）。正規表現にチェックすると 検索 欄を正規表現として扱います（フラグは既定で g）。まず「プレビュー」で件数を確認してから「実行」してください。
      ※ 元に戻せません。実行前のバックアップを推奨します。ルールはこの端末に保存されます。

      外部LLMで取り込む

      claude.ai・ChatGPT・Gemini など任意のチャットAIで取り込めます。Anthropic APIキー不要。
      手順: ①下のプロンプトをコピー → ②チャットAIにPDF（または画像）と一緒に貼り付けて送信 → ③返ってきたJSONを下の欄に貼り付け → ④「読み込む」で編集画面へ。

      ストップワードリスト

      リストは Worker(D1) に保存され全端末で共有されます。「内蔵」は編集できません。

      レベル別語彙リスト（CEFR分析）

      各行に「語 レベル」（例: abandon B2）を記入。レベルは A1/A2/B1/B2/C1/C2。Worker に保存され全端末で共有されます。既定で Oxford Word List 5000 を内蔵。
